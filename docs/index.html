🐴 O Problema

O povo brasileiro foi convencido com pão com mortadela de que a picanha voltaria à mesa. Quatro anos depois: nem a mortadela está barata. A blockchain não mente. Os políticos, sim.

🍖 A Solução

A Mulacoin não promete picanha. Ela entrega 13 FazoL por bloco, todo minuto, sem parar. Sem palanque. Sem passeata. Sem patrão. Descentralizada de verdade.

🔢 Por que 13?

O número 13 é a recompensa por bloco. Coincidentemente, é também o número do partido que prometeu a picanha. Cada bloco minerado é um lembrete carinhoso.

💰 Por que FazoL?

O L do FazoL é uma homenagem ao slogan de campanha que viralizou nas redes. Feijão com L. Uma moeda humilde, do povo, pra quem ainda sonha com a picanha.

Fork do Dogecoin Core com algoritmo Scrypt PoW. Minerável por CPU comum. Carteira em português. Endereços começam com "M" — de Mula, de Mortadela, de Memória.

📜 Sem Promessas

A Mulacoin é um projeto de humor político. Não prometemos valorização, não prometemos picanha. Prometemos apenas que a blockchain vai continuar rodando — diferente de certas promessas eleitorais.

COMO
MINERAR

Mineração por CPU com o cpuminer. Sem placa de vídeo cara, sem ASIC. Qualquer computador pode participar da rede e ganhar seus 13 FazoL por bloco.

01

Baixe e instale a carteira

Baixe o binário para seu sistema operacional, extraia e execute o mulacoin-qt. A carteira vai sincronizar automaticamente com a rede via DNS seed.

02

Gere um endereço de recebimento

Na carteira, vá em Arquivo → Receber pagamentos e copie seu endereço (começa com "M"). É para esse endereço que as recompensas de mineração vão.

03

Instale o cpuminer

No Linux, clone e compile o cpuminer:

git clone https://github.com/pooler/cpuminer
cd cpuminer && ./autogen.sh && ./configure && make
04

Inicie a mineração

Com a carteira aberta e sincronizada, execute o minerd apontando para ela:

./minerd -a scrypt -o http://127.0.0.1:23560 \
-u SEU_USUARIO -p SUA_SENHA \
--coinbase-addr=SEU_ENDERECO_M -t 2

Substitua SEU_ENDERECO_M pelo endereço copiado no passo 2. O parâmetro -t 2 define o número de threads (núcleos da CPU).
